# 争端分歧机 (Dispute Machine)

> 一个解决争端/矛盾/决策的Web工具，把矛盾从"吵架"变成"规则化对抗"

## 产品定位

- **目标用户**：各种矛盾决策场景（朋友矛盾、工作决策、家庭琐事）
- **产品风格**：轻松但不儿戏，有仪式感
- **平台**：H5移动端（后续支持小程序）

## 功能版本

### 1.0 运气服人版
- 两人争端：剪刀石头布
- 多人争端：比大小（骰子、抽卡、写数字等多种玩法）
- 结果随机但公平，解决日常小事

### 2.0 以德服人版
- 说服型：双方/多方提交理由，打分，分高者胜
- 讨论型：多方讨论细节，打分投票
- 支持匿名打分、权重设定、协议签署

## 技术栈

- **前端**：Vue 3 + Vite + Vant 4
- **后端**：Node.js + NestJS
- **数据库**：SQLite
